### 工具

1. Memory Profiler

    Android Studio自带的工具，使用图形界面展示，比较方便看，可以直观的看到内存变化、内存抖动、泄漏等
    
    这里可以捕获堆转储，GC，跟踪内存分配等

2. Memory Analyzer

3. LeakCanary


### 优化

1. 内存大小：峰值越低越好，需要优化前后做对比

2. 内存泄漏：可用内存减少，需要用工具检查对比优化前后

3. 内存抖动：频繁GC

4. 内存溢出：oom
