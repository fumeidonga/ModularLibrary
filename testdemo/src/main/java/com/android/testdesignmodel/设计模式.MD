### 2. 为什么要用设计模式 ###
设计模式是某类特定问题的代码设计解决方案
是一套代码设计的经验总结，使用设计模式我们可以：
* 提高代码复用率，降低开发成本和周期
* 提高代码可维护性、可拓展性
* 使代码更加优雅
* 让代码更容易被他人理解
